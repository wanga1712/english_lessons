# Размеры карты и локаций для выбора спрайта героя

## Размеры локации (острова)

- **Радиус острова:** 50 пикселей
- **Диаметр острова:** 100 пикселей (50 × 2)
- **Форма:** Круг

## Размеры карты

- **Ширина карты:** 2000 пикселей
- **Высота карты:** 2000 пикселей
- **Область видимости:** Зависит от размера окна браузера (обычно 800-1600px)

## Рекомендуемый размер героя

### Для локации диаметром 100px:

**Оптимальный размер спрайта героя:**
- **Высота:** 80-120 пикселей
- **Ширина:** 40-60 пикселей
- **Формат:** PNG с прозрачностью

### Почему такой размер?

1. **Герой должен быть виден** на локации размером 100px
2. **Герой должен быть крупнее локации** (как в Heroes 3)
3. **Герой должен помещаться** на экране при зуме

### Примеры размеров:

| Размер спрайта | Как будет выглядеть |
|----------------|---------------------|
| 64×64 px | Маленький, но видимый |
| 96×96 px | **Оптимальный** (почти как локация) |
| 128×128 px | Крупный, хорошо виден |
| 192×192 px | Очень крупный (может быть слишком большим) |

## Текущие настройки в коде

```javascript
const islandRadius = 50; // Радиус локации = 50px
// Диаметр = 100px

// Герой стоит на 60px выше центра локации
heroContainer.y = pos.y - 60;
```

## Рекомендации для выбора спрайта

### ✅ Идеальный спрайт:
- **Размер:** 96×96 или 128×128 пикселей
- **Стиль:** Изометрический или вид сверху (как Heroes 3)
- **Прозрачность:** PNG с alpha-каналом
- **Ориентация:** Лицом к камере или под углом 45°

### ⚠️ Что учесть:

1. **Масштабирование:** Phaser может масштабировать спрайт
   ```javascript
   hero.setScale(0.5); // Уменьшить в 2 раза
   hero.setScale(1.5); // Увеличить в 1.5 раза
   ```

2. **Анимации:** Если есть спрайт-лист с анимациями:
   - Idle (покой): 4-8 кадров
   - Walk (ходьба): 4-8 кадров
   - Attack (атака): 4-6 кадров

3. **Позиционирование:**
   - Герой должен стоять на локации (ноги на острове)
   - Центр спрайта должен быть внизу (origin: 0.5, 1)

## Пример использования в коде

```javascript
// Загрузка спрайта 96×96
this.load.image('hero', 'path/to/hero-96x96.png');

// Создание героя
const hero = this.add.image(x, y, 'hero');
hero.setScale(1); // Без масштабирования (96×96)
hero.setOrigin(0.5, 1); // Якорь внизу (ноги на локации)
hero.setDepth(15); // Выше всего
```

## Где искать спрайты такого размера

### OpenGameArt.org:
- Фильтр: Размер → 64×64, 96×96, 128×128
- Категория: Characters → Fantasy, RPG

- "96x96 hero sprite"
- "128x128 fantasy character"
- "isometric hero 96px"
- "RPG character sprite sheet"

**Выбирайте спрайт размером 96×96 или 128×128 пикселей** - это будет идеально смотреться на локации диаметром 100px и будет хорошо виден на карте.
